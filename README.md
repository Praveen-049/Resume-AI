# Candidate Ranking AI

A full-stack AI-powered candidate ranking platform for matching resumes to job descriptions using embeddings, ranking models, and a web interface.

## Project Structure

- frontend/: React-based UI
- backend/: FastAPI backend service
- data/: Sample candidates, jobs, and processed data
- ml/: Training, inference, and evaluation modules
- docker/: Containerized infrastructure stack
- docs/: Architecture and setup documentation

## Quick Start

1. Clone the repository
2. Start infrastructure services with Docker Compose
3. Run the backend
4. Run the frontend

## Tech Stack

- Frontend: React + Vite + Tailwind
- Backend: FastAPI + Python
- Search/Vector DB: Qdrant
- Storage: PostgreSQL
- Messaging: Kafka
- Cache: Redis
- Experiment Tracking: MLflow
